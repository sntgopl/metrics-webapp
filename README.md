## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

>### Prerequisites
 - CMD
 - Node.js
>### Setup

- $git clone https://github.com/sntgopl/metrics-webapp.git
- Access `cd financial-modelling`


>### Install

- Write `npm install`

>### Usage

- Execute `npm run`

>### Run tests

- Execute `npm test`
